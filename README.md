```markdown
# Электронный журнал для оценок учеников

Этот проект представляет собой электронный журнал для отслеживания оценок учеников. 
Он создан с использованием FastAPI и SQLAlchemy.

## Эндпоинты

### Ученик (Student):

- **POST /students/**  
  Создать нового ученика.

- **GET /students/{student_id}**  
  Получить информацию о конкретном ученике.

- **PATCH /students/{student_id}**  
  Обновить информацию о конкретном ученике.

- **DELETE /students/{student_id}**  
  Удалить ученика.

### Оценка (Score):

- **POST /score/**  
  Добавить оценку ученику.

- **GET /score/{score_id}**  
  Получить информацию о конкретной оценке.

- **PATCH /score/{score_id}**  
  Обновить информацию о конкретной оценке.

- **DELETE /score/{score_id}**  
  Удалить оценку.

## Коллекция Postman

В папке `collection` можно найти Postman коллекцию для тестирования.

## Запуск приложения

Для запуска приложения необходимо:
1. Перейти в папку `gradebook/app`.
2. Запустить команду:
   ```
   fastapi dev main.py
   ```
```
